# Учет приборов

Веб-приложение для учёта приборов с хранением данных в Supabase (PostgreSQL).

## Функционал
- Авторизация (admin / employee)
- Добавление, редактирование, удаление приборов
- Выдача, возврат, передача, бронирование
- Календарь поверок, выдач и бронирований
- Фильтрация по поверке, состоянию, пользователю
- Массовые операции (списание, удаление) для администратора
- Тёмная тема
- QR-коды для приборов
- Комментарии

## Настройка
1. Создайте проект в Supabase и выполните SQL-скрипт для создания таблиц.
2. В `config.js` укажите `supabaseUrl` и `supabaseAnonKey`.
3. Загрузите файлы на GitHub Pages или любой хостинг статики.

## Технологии
- HTML/CSS/JS (без фреймворков)
- Supabase (PostgreSQL)
- QRCode.js
